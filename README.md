# Caltrans Road Safety Analysis (2025)

## Project Overview
This project analyzes real-world traffic data through 2025 to identify high-risk patterns and provide actionable safety recommendations for Caltrans management. The goal is to reduce fatal injuries and improve road infrastructure efficiency.

## Technical Workflow
* **Data Cleaning:** Processed and structured raw data using **Excel** to ensure accuracy.
* **Data Retrieval:** Leveraged **SQL** to query specific safety metrics and traffic patterns.
* **Visualization:** Developed interactive sheets in **Tableau** to identify peak accident hours, dangerous road types, and primary crash causes.

## Key Insights & Recommendations
* **Peak Risk:** Identified a significant surge in accidents during the 5:00 PM rush hour.
* **Infrastructure:** Found that "Not Divided" roads and "Four-Way" intersections account for the highest volume of fatal injuries.
* **Safety Strategy:** Proposed the implementation of plastic dividers and "Road Zippers" (similar to the Golden Gate Bridge) to prevent head-on collisions.
